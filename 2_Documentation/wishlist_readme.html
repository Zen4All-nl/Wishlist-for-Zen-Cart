PURPOSE & AIM

Wishlist is an module that will allow your customers to put their favorite products on a wishlist. They can also share the list(s) with friends by sending an email with the link to it.

Version 0.9 adds attribute support
Version 0.8 fixes some bugs found related to new functionality.
**Update: Version 0.7 is a speeded release to fix problem with button to add to cart from the list.**
Version 0.6 is based on v0.5 of Wishlist but has been undergoing MAJOR modifications especially in the area of renaming files and fixing of minor yet annoying bugs

If you are allready using a previous version of the wishlist v0.5 or lower, do read very carefully how to delete your files.

If you are starting a clean install you should be ready to go in a matter of minutes!

Wishlist v0.6 was tested for Zen Cart 1.3.9h and PHP 5.3. It may work on other versions (don't see why not) but this hasn't been tested yet

After following the installation instructions, you will need to read the "How to Use" section. There you may pick up some hints how you can integrate Wishlist nicely into your site. (enable sidebox, create links to wishlist module in core-files)

INSTALLATION

!!!!!BACKUP YOUR STORE BEFORE YOU BEGIN!!!!!

There are THREE folders in the fileset

  • readme (you allready found it, good)
  • admin
  • includes

Note: If you have a version 0.5 or older, you will need to delete the files that were installed with this version
Do not start the installation if such an older installation exists. Look at the uninstall instructions below if this applies to your situation.

Basic Installation Instructions
  1. -----[ 1. Create tables ]----- Run 'wishlist.sql' in the Install SQL patches in the Zen Cart admin. Important: If you use a database table prefix (see file "includes/configure.php", line 52), and you want to install the tables directly via phpadmin then ammend the statements as they should be. Running the install file via the zen cart sql patches tool within zen-cart works fine without amendments and is preferred.
  2. -----[ 2. Upload ]----- Rename the "admin" folder to the name of your admin folder. Rename the "includes/templates/YOUR_ZEN_TEMPLATE" folder to the name of your template. Upload the files found in the "includes" directory of this distribution to their corresponding location of your ZenCart implementation. The same for the files in the renamed admin folder
  3. -----[ 3. Insert button on product page ]----- In the includes\templates\YOUR_ZEN_TEMPLATE\templates you will find an example file which includes an example of how you could integrate a button on your products page. You need to ammend your original tpl_product_info_display.php. Put the code whereever you would like the wish button to appear but be aware it need to be within the form cart_quantity to include attributes. (also see HINTS to find out how you can add a link in your header toolbar)
  4. -----[ 4. Activate sidebox (optional) ]----- If you would like the sidebox, then... Login to the ZenCart admin. Select "Tools -> Layout Boxes Controller". Select "sideboxes/wishlist.php" in the box list. Turn "Column Status" ON. Save changes. Important! the sidebox does NOT include support for attributes. from this version onwards i would discourage use of this sidebox the sidebox may return with improved code but at this time i would recommend only using a link in your product page
  5. -----[ 5. side specific css/formats (optional) ]----- you will want to match the add to wish list button to match your site edit includes\templates\YOUR_ZEN_TEMPLATE\buttons\english (or your language folder) to match in a button that is used on your site also look at includes\templates\YOUR_ZEN_TEMPLATE\css\style_un_classic.css if you don't like the layouts in this contribution edit to match your site's layout
Upgrading from previous versions of Wishlist
  1. There is no upgrade path if you are using version 0.5 or older
  2. You will need to uninstall the older distribution before you can install version 0.6
  3. However: you will not loose the existing data about the wishlists which are stored in the database. After deinstalling/installing v0.6 all wishlist will be available again to your customers
  4. version 0.9 adds an extra column to the wishlist products table, this column will be empty for existing wishlists.

HOW TO USE

After the install, altough the module is enabled by default you will see no changes in your store front.

This is because you will most likely want to place files next to other buttons that are in use and these edits are all in zen-cart core files which we donot want to overwrite for obvious reasons. If you want a quick look and feel of the module, go inside your layout boxes controller and enable the sidebox. It will only show on pages with a product and contains a link to the wishlist page in the header by default.

Be aware that the sidebox is not supporting attributes and the sidebox may be removed (or improved) in future versions. Using sidebox on live site is not recommended at this time.

Admin Options

This version includes an admin option to enable/disable the module and more. Look for "Wish list" under the configuration menu of your store to look at the available options. The default settings should be good for most store setups.

To include an "Add to Wish List" link on a product template


<!--bof Wishlist button -->
<!--<br class="clearBoth" />
optional depending where the code is added-->

<?php if (UN_MODULE_WISHLISTS_ENABLED) {
if ( $_SESSION['customer_id'] ) {
echo '<div id="productWishlistLink" class="buttonRow back">';
echo zen_image_submit(UN_BUTTON_IMAGE_WISHLIST_ADD, UN_BUTTON_WISHLIST_ADD_ALT, 'name="wishlist" value="yes"');
echo '</div>';
} else {
echo '<div>';
echo UN_BOX_WISHLIST_LOGIN_TEXT;
echo '</div>';
}
} else {}
?>
<!--eof Wishlist button -->

To include an "Add to Wish List" link on a non-product template, try this (replace YOUR_PRODUCTS_ID with a valid product id):


<?php
// start wishlist link
if (UN_MODULE_WISHLISTS_ENABLED) {
echo '<a href="' . zen_href_link(UN_FILENAME_WISHLIST, 'products_id=' . YOUR_PRODUCTS_ID . '&action=wishlist_add_product') . '">' . UN_TEXT_ADD_WISHLIST . '</a>';
}
// end wishlist link
?>

To include a "Wish List" link in the template header


<?php if (UN_MODULE_WISHLISTS_ENABLED) { ?>
<a href=""><?php echo UN_HEADER_TITLE_WISHLIST; ?></a>
<?php } else {} ?>
(see also the example file in the contribution /includes/templates/YOUR_TEMPLATE/common/EXAMPLEtpl_header.EXAMPLE)
